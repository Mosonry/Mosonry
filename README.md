### 🧭 技术定位

在学从底层硬件驱动到上层服务器运维的完整实践。以 **嵌入式开发** 为核心，正向 **机器视觉** 与 **边缘计算** 延伸。

- **核心领域**：STM32 开发、C 语言模块化编程、传感器驱动、Linux 基础运维
- **基础设施**：独立维护个人云服务集群 (Docker, Nginx, Cloudflare, 1Panel)
- **探索方向**：Python 生态、OpenCV 机器视觉、轻量级机器学习模型部署
